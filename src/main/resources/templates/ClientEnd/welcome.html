Welcome to BruteForces !

This is a project for online judge system based on Java/SpringBoot framework.

Here is the guide for you for the first time using BruteForces or any online judge system.

What can you do in our website:

  • (Problems) Submit your code and get judged.
  • (Contests) Compete with other programmers.
  • (Status) Record your submission every time.
  • (Blogs) Learn new algorithm by read blogs.
  • (OnlineIde) Code online and get judged.

Here is the description of result after the judgement:

Result Description
Accepted Your submission is right.
Compile Error Your code get errors when compiling.
Wrong answer The output of you code is wrong.
Time Limit Exceed You code can not output answer on time.
Memory Limit Exceed You code spend more memory than the limitation.
Runtime Error You code get errors when running.
